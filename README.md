# project-superstore-datasets-
This project analyzes the Superstore dataset using Python and Pandas to understand sales, profit, and business performance
Python,Pandas,Matplotlib are tools used
 
 several steps are used like..
,Data loading and cleaning
,Feature engineering
,Data analysis (groupby, pivot tables)
,Data visualization
   
    Key Insights
Technology is the most profitable category

Some regions have low profit

Sales and profit have a weak positive relationship

Sales show seasonal trends

...The analysis helps identify patterns and supports better business decisions.
